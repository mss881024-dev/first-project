# Windsurf

## Setup

1. Download Windsurf IDE from [codeium.com/windsurf](https://codeium.com/windsurf)
2. Open this repository as a workspace
3. Cascade (AI panel) activates automatically

## Key Features
- Cascade: agentic multi-file editing with context awareness
- Flows: automated multi-step coding workflows
- Deep codebase indexing for accurate autocomplete
- Integrated terminal and diff viewer

## Configuration
- **Workspace rules**: `.windsurf/rules.md` — loaded into every Cascade session
- **Prompts**: `prompts/windsurf/`

## Tips
- Reference `docs/workflows/` inside Cascade for complex tasks
- Use `prompts/windsurf/` to save effective Cascade instructions
- Cascade respects `.gitignore` when indexing
